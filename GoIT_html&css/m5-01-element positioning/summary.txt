Module 05.1 позиционирование элементов
	1. Есть поток документа, а есть некоторые css свойства, которые из потока документа вынимают элементы, подымают над потоком документа. Это float например. Так вот float не один, есть еще специальное свойство position, который так же может вынимать элементы из потока документов.
	2. Что такое позиционирование? Это гибкий механизм расположения документов. Позиционирование используется, когда мы не можем что-либо реализовать используя поток документов.
	3. Отвечает за позиционирование свойство position, которое устанавливает режим позиционирования. Есть 4 значения: top, bottom, left, right - указывают координаты в которых мы хотим спозиционировать  элементы.
	4. z-index - управление порядком слоев.
	5. Ось x - позволяет позиционировать слева на право.
	6. Ось y - позволяет позиционировать сверху вниз.
	7. z-index - это от наших глаз до монитора, 3D. Как в photoshop слои.
	8. Какие есть режимы позиционирования:
		▪ position: static; /* по умолчанию */
		▪ position: relative; /* относительное */ Когда элемент позиционируется относительно своей стартовой позиции.
		▪ position: absolute; /* абсолютное */ Позволяет вынуть документ из потока документов.
		▪ position: fixed; /* фиксированное */ Когда можно на странице зафиксировать элемент относительно окна. И при прокрутке скролом он находиться в фиксированном положении.
		▪ position: sticky; /* закреплённое */ Гибрид relative и fixed.
	9. Relative - когда элементу дается relative, то смещение элемента происходит относительно своего элемента.
