Module 05.1 позиционирование элементов

	1. Есть поток документа, а есть некоторые css свойства, которые из потока документа вынимают элементы, подымают над потоком документа. Это float например. Так вот float не один, есть еще специальное свойство position, который так же может вынимать элементы из потока документов.
	2. Что такое позиционирование? Это гибкий механизм расположения документов. Позиционирование используется, когда мы не можем что-либо реализовать используя поток документов.
	3. Отвечает за позиционирование свойство position, которое устанавливает режим позиционирования. Есть 4 значения: top, bottom, left, right - указывают координаты в которых мы хотим спозиционировать  элементы.
	4. z-index - управление порядком слоев.
	5. Ось x - позволяет позиционировать слева на право.
	6. Ось y - позволяет позиционировать сверху вниз.
	7. z-index - это от наших глаз до монитора, 3D. Как в photoshop слои.
	8. Какие есть режимы позиционирования:
		▪ position: static; /* по умолчанию */
		▪ position: relative; /* относительное */ Когда элемент позиционируется относительно своей стартовой позиции.
		▪ position: absolute; /* абсолютное */ Позволяет вынуть документ из потока документов.
		▪ position: fixed; /* фиксированное */ Когда можно на странице зафиксировать элемент относительно окна. И при прокрутке скролом он находиться в фиксированном положении.
		▪ position: sticky; /* закреплённое */ Гибрид relative и fixed.
	9. Relative - когда элементу дается relative, смещение элемента происходит относительно своего положения.
	10. Значение relative для свойства position позволяет элементам отображаться в пределах обычного потока документа, оставляя пространство для элемента по учмолчанию и не позволяя другим элементам его обтекать; однако оно позволяет изменять положение отображения элемента с помощью свойств смещения left|right|top|bottom.
	11. Сам по себе position: relative практически бесполезен и никогда не используется, но он очень важен в комбинации с position: absolute, поэтому важно знать его поведение.
	12. Значение absolute для свойства position отличается от relative тем, что элемент с абсолютным значением position не будет отображаться в нормальном потоке документа, а исходное пространство и позиция, абсолютно позиционированного элемента, не будут сохранены.
	13. position: absolute вынимает из потока документов подобно float и просто над документом элемент висит, а все под него заезжает.
	14. Position: absolute
		a. Элемент выпадает из потока.
			- Сжимается под содержимое.
			- Гибкая модель позиционирования.
			- Свойства задания координат ведут себя по-другому.
			- Участвует в «наслаивании».
		b. Положение по умолчанию
			- Если абсолютному элементу не задавать никаких координат, то он привязывается левым верхним углом к той точке, где был бы, если бы был нормальным.
			- Если эта точка смещается — он смещается вместе с ней.
			- Из этой точки его можно смещать внешними отступами.
		c. Координаты абсолютного элемента
			- Привязка к верхнему левому углу
			- Привязка к правому нижнему углу
		d. Изменение точки привязки
			- Если среди родителей абсолютного элемента есть тег с относительнымпозиционированием, то абсолютный привязывается к нему.
			- Если относительных родителей несколько, то выбирается ближайший.
			- А если родителей с относительным позиционированием нет,  то блок позиционируется относительно body.
		e. Относительные координаты %
			- относительно окна
			- относительно родителя
		f. Центровка
			- Если знаем ширину и высоту то отрицательный маржин
			- Если нет то transform
		g. Относительные размеры
			- Привязка происходит не только для точки отсчёта координат, но и для относительных размеров.
			- /* ширина и высота абсолютного равны половине ширины и высоты родителя */
			- position: absolute;
			- width: 50%;
			- height: 50%;
			- transform: translate(-50%, -50%);
	15. Правило: абсолютно позиционированный элемент для него свойство позиционирования: top, right, left, bottom работают относительно границ родителя, относительно которого он абсолютно позиционирован.
	16. Фиксированное позиционирование position: fixed; Позиционирует элемент относительно вьюпорта. Элемент, как будто прибит к определенному месту на вьюпорте.
		a. Похоже на абсолютное позиционирование.
		b. Отсчёт координат всегда привязан к окну браузера.
		c. Элемент «прибивается» к точке экрана и не смещается даже при прокрутке.
		d. Участвует в «наслаивании».
	17. Кроме того, абсолютно позиционированные элементы перемещаются относительно их ближайшего родительского элемента с position отличным от static. Если такой родительский элемент не существует, абсолютно позиционированный элемент будет располагаться относительно элемента body.
	18. Управление порядком слоёв - z-index:
		a. У кого больше индекс, тот и выше.
			- Выше тот, кто дальше в коде, если индекса нет или он одинаковый.
			- Индекс работает только для относительных, абсолютных и фиксированных элементов.
			- Если position: static; работать не будет.
	19. Как это читать? .card:hover .social-icons {  opacity: 1; } Вот это CSS правило для social-icons применить тогда, когда происходит эффект hover на карточку.
	20. Position: absolute работает относительно любого элемента, у которого позиционирование абсолютное от static.
