Module 03 Блочная модель
	1. У CSS есть свойство display. У display есть разные значения, но 3 самых важных
		a. display: block - знак переноса есть всегда --> <div>, <h1 … h6>, <p>.
		b. display: inline-block -  знак переноса по необходимости
		c. display: inline - знак переноса по необходимости --> <span>, <a>, <img>.
	2. Блочный элемент занимает максимальную ширину страницы, а высоту он занимает по контенту по умолчанию.
	3. Строчный элемент занимает ширину и высоту по контенту.
	4. <div> - блочный элемент по умолчанию, но его всегда можно переопределить.
	5. Блочные элементы нельзя помещать внутри строчных. Исключение, если внутри ссылки поместить <p> - валидатор пропустит. Валидатор устал писать сообщения и это выключили, но по спецификации нельзя.
	6. display: block. Блочные элементы занимают всю ширину страницы, имеют перед и после знак перенос, высоту занимают по контенту. За ним невозможно что-либо поставить, так как он занимает всю ширину страницы по умолчанию. Может и не всю ширину занимать, если проставить размеры, но наличия знаков переноса не позволяют поставить что-либо за ним.
	7. display: inline-block. Это тот же блок, ему можно задать высоту и ширину, но у него нет знаков переноса.
display: inline. Нет знаков переноса, но и нет возможности задавать высоту и ширину.
